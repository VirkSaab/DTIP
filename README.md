## How to Install
### Prerequisites to install Python 3.8 virtual env
```bash
apt update
apt install software-properties-common
add-apt-repository ppa:deadsnakes/ppa
apt update
apt install python3.8 python3.8-venv python3.8-dev
```
### Create a virtual env
```bash
make make_venv
```
